# 文明策略 (Civilization Strategy) - 游戏指南

## 游戏概览

这是一个 Paradox 风格的复杂策略游戏，模仿《哈耶克的文明：市场经济》。玩家管理一个国家，需要平衡经济、人口、军事、科技和外交。

## 核心系统

### 1. 资源管理
- **金币**: 用于招募军队、建造建筑、外交活动
- **食物**: 用于养活人口，决定人口增长
- **生产**: 用于加快建筑建造速度
- **科学**: 用于研究新技术
- **文化**: 增加民众满意度和国家影响力

### 2. 人口系统
人口分为5个阶级，每个都有特殊作用：
- **贵族**: 提升外交能力
- **商人**: 增加贸易收入（最多）
- **农民**: 增加食物产量
- **工人**: 增加生产和科学产出
- **士兵**: 增加军事力量（消耗资源）

### 3. 经济系统
- **税率**: 在概览面板调整，高税率增加金币但降低民众满意度
- **建筑**: 每个建筑产生资源或奖励
- **贸易**: 与其他国家建立贸易关系
- **市场**: 资源价格随供求变动

### 4. 科技树
4个时代，18项技术：
- **古代**: 农业、青铜、文字、数学、砌体
- **古典**: 铁器、哲学、骑术、货币、戏剧
- **中世纪**: 封建、教育、骑士精神、银行、火药
- **文艺复兴**: 印刷、科学方法、重商主义、民族主义

每项技术有前置要求，解锁新建筑和能力。

### 5. 建筑与建造
24种建筑分为5类：
- **经济**: 农场、市场、银行、工坊、工厂
- **科技**: 图书馆、大学、学院
- **文化**: 纪念碑、寺庙、剧院、博物馆
- **军事**: 营房、马厩、要塞、军械库
- **民生**: 仓库、渠道、住房、孤儿院

建筑需要满足科技前置要求。

### 6. 军事系统
- **单位类型**: 民兵(1战力)、步兵(3)、骑兵(4)、炮兵(6)
- **招募**: 在军事面板选择单位类型和数量
- **维护**: 每回合消耗金币
- **战争**: 可向其他国家宣战或提议和平

### 7. 外交系统
与4个AI国家互动：
- **北方帝国** (好战): 倾向发动战争
- **商业共和国** (贸易): 重视贸易和合作
- **东方王国** (平衡): 比较温和
- **南方联邦** (和平): 倾向和平与文化

可以提议联盟、贸易协议或送礼改善关系。

### 8. 事件系统
每回合有40%概率触发随机事件，包括：
- 丰收、饥荒、瘟疫
- 贸易机遇、犯罪浪潮
- 科学发现、民众叛乱

每个事件有3个选择，影响游戏状态。

## 游戏流程

1. **查看当前状态**: 在概览面板查看国家各项指标
2. **管理资源**:
   - 调整税率以平衡收入和民众满意度
   - 建造建筑以增加资源产出
   - 建立贸易关系
3. **研究科技**: 在科技面板选择要研究的技术
4. **建造建筑**: 在建筑面板选择要建造的建筑
5. **管理人口**: 监控人口增长、满意度、稳定性
6. **军事建设**: 招募军队、建造防御工事
7. **外交活动**: 与其他国家建立关系、提议条约
8. **推进回合**: 点击"下一回合"按钮

## 胜利条件（可扩展）
目前游戏是无限进行的沙盒模式，可自定义胜利条件：
- 达到一定的总人口
- 研究所有科技
- 建立与所有国家的联盟
- 积累一定的财富

## 提示

1. **平衡是关键**: 不要过度征税，会导致人口不满
2. **多样化经济**: 不要只依赖一种资源，建造各种建筑
3. **科技优先**: 科技是进步的基础，定期研究新技术
4. **人口管理**: 保持食物充足以支持人口增长
5. **外交优势**: 联盟可以提供安全感，贸易增加收入
6. **应对事件**: 仔细选择事件的解决方案，考虑长期影响
7. **军事防守**: 保持一定的军事力量以防备来犯

```
│   ├── PopulationSystem.js     # 人口系统
│   ├── MilitarySystem.js       # 军事系统
│   └── DiplomacySystem.js      # 外交系统
├── data/             # 游戏数据
│   ├── technologies.js         # 科技定义
│   ├── buildings.js            # 建筑定义
│   ├── events.js               # 事件定义
│   └── nations.js              # AI国家定义
├── components/       # React组件
│   ├── GameApp.jsx             # 主游戏组件
│   ├── TopBar.jsx              # 资源条
│   ├── OverviewPanel.jsx       # 概览面板
│   ├── EconomyPanel.jsx        # 经济面板
│   ├── PopulationPanel.jsx     # 人口面板
│   ├── TechPanel.jsx           # 科技面板
│   ├── BuildPanel.jsx          # 建筑面板
│   ├── MilitaryPanel.jsx       # 军事面板
│   ├── DiplomacyPanel.jsx      # 外交面板
│   ├── EventPopup.jsx          # 事件弹窗
│   └── GameLog.jsx             # 游戏日志
├── App.jsx           # 应用入口
├── main.jsx          # React挂载点
└── index.css         # Tailwind配置
```

## 技术栈

- React 18
- Vite
- Tailwind CSS v4
- Lucide React (图标)

## 部署

项目配置为在 `/civ-strategy/` 路径下部署到GitHub Pages。

```bash
npm run build
# 输出到 dist/ 目录
```
